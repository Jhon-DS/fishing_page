# 🎣 Simulação de Página de Phishing

Este projeto foi desenvolvido como um **exercício educacional** proposto durante o curso Técnico em **Desenvolvimento de Sistemas na ETEC**.

O objetivo do desafio foi criar uma **simulação de páginas de phishing**, reproduzindo visualmente páginas utilizadas em ataques de engenharia social, com a finalidade de compreender **como esses golpes funcionam e como identificá-los**.

⚠️ **Importante:** Este projeto possui finalidade **exclusivamente educacional**, voltado para o aprendizado sobre segurança na internet.

---

# 🎯 Objetivo do Projeto

O projeto tem como objetivo demonstrar como páginas falsas podem ser criadas para **imitar serviços reais** e induzir usuários a fornecer informações sensíveis.

Com isso, o exercício ajuda estudantes a:

- compreender técnicas de **engenharia social**
- identificar **padrões comuns em páginas fraudulentas**
- entender como **redirecionamentos podem ser usados em golpes**
- desenvolver maior **consciência sobre segurança digital**

---

# 🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias básicas de front-end:

- **HTML**
- **CSS**
- **JavaScript**

### Redirecionamento Aleatório
Ao clicar no botão **"comprar"**, o usuário é redirecionado **aleatoriamente para uma das três páginas de phishing** presentes no projeto.

Esse comportamento simula como ataques reais podem utilizar múltiplas páginas falsas para capturar informações.

## 📝 Considerações

Este é um projeto simples, desenvolvido com foco no aprendizado e na prática de conceitos básicos de HTML, CSS e JavaScript, além da compreensão de técnicas utilizadas em engenharia social.

Apesar de sua simplicidade, o exercício contribui para desenvolver uma visão mais crítica sobre segurança digital e os riscos presentes na navegação na internet.
